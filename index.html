Acerca de mi :)

Hola! mi nombre es Consuelo y soy ingeniera en biotecnología. Me gusta estar constantemente aprendiendo cosas nuevas y es por esto que tengo muchos hobbys (tejer, bordar, cocer, cocinar y jardinear… a futuro me gustaría aprender a hacer cerámicas). Durante un largo tiempo me dedique a la investigación científica, pero eventualmente decidí cambiar el rumbo de mi desarrollo profesional, y empecé a aprender desarrollo web fullstack.

Mis trabajos

Aquí puedes encontrar los trabajos que he realizado durante el primer módulo del bootcamp en Desafío Latam. Haciendo click en "Ver más" podrás acceder al repositorío en GitHub de cada trabajo.

Desafio - Mi Curriculum Vitae en HTML

Primer control del bootcamp sobre la clase de HTML, aquí se evaluó el conocimiento de la estructura de lenguaje de programación y el correcto uso de etiquetas semánticas.

Desafío 2 - Construyendo un landing page

Segundo control del bootcamp sobre la clase de CSS, en este desafío se evaluó el poder agregar estilos al HTML que creamos.

Desafío 3 - Iguana page

Tercer control donde se evaluó la clase de posicionamiento con CSS, en este control se entregó una maqueta la cual se debía seguir para construir la página.

Desafío 4 - Cuppon

Cuarto control del primer modulo del bootcamp, se evaluó lo aprendido en la clase de Bootstrap, como implementarlo y modificarlo en nuestro proyecto. Este proyecto además es el primero en ser un HTML responsivo utilizando clases de bootstrap.

Desafío 5 - Github

Quinto y último control del módulo de introducción al desarrollo web. En este trabajo se evaluó el correcto uso de Git y GitHub, para lo cual se hizo un fork de un proyecto que se encontraba en GitHub y se realizaron modificaciones al código.
